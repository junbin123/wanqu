# 使用 BeautifulSoup 爬取湾区日报文章

> 本教程针对的是爬虫新手，照着教程一步一步操作，一定能够实现，对于入门是非常友好的教程！

教程大致可分为以下步骤：

1. 连接获取网页内容
2. 使用 BeautifulSoup 获取 html 内容
3. 在 soup 对象里的循环搜索 HTML 元素
4. 进行简单的数据清理
5. 把数据写入 csv 文件中
